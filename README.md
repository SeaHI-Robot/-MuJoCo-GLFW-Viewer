## A Simple MuJoCo GLFW Viewer

> Based on MuJoCo Python Binding and GLFW



<br>



### Usage

- Implement user controller in `controller_implement.py`
- Run `main.py`

### Characteristics
1. Adjust camera view (in `main.py`)
2. Visualize contact (in `main.py`)
3. Mouse interaction.
